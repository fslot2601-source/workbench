# Workbench 1.2.2

这个版本调整了状态栏面板的空间使用。

## 本次更新

- 状态栏面板会根据当前实际内容自动调整高度，不再固定保留一大块空白。
- 未显示 Token 活动或本机存储时，面板会随之收缩。
- 内容较多时，面板保持合理的最大高度，并通过滚动继续查看全部内容。
- 缩小分区、卡片和操作按钮之间的纵向间距，整体更接近原生 macOS 工具面板。

## 系统要求

- macOS 14 或更高版本。
- 本机已安装并登录 ChatGPT 桌面版，或已单独安装 Codex CLI。

## 下载

- `Workbench-1.2.2-15.dmg`：拖入“应用程序”安装。
- `Workbench-1.2.2-15.zip`：直接解压使用。
- `Workbench-1.2.2-15-SHA256SUMS.txt`：下载校验值。

当前社区构建没有 Apple Developer ID 签名，也没有经过 Apple 公证。首次打开时，请在 Finder 中右键 Workbench 并选择“打开”。
